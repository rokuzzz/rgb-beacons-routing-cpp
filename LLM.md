AI/LLM use in course
====================

From University rules:

> "Artificial intelligence can be used in the creation of outputs, but the
> student must clearly report its use.  Failure to disclose the use of AI will
> be interpreted as fraud. The use of AI may affect the assessment."
   
Please fill out following items OR replace this file with text 
"I did not use AI/LLM in creation of this project"

- In what parts of the course projects LLM tool was used
- Which LLM tools were used
- Motivation/reason to use LLM
- How and why LLM helped
- What kind of mistakes LLM did
- What were things that LLM was not able to provide

This report will be used in grading and we may use it (after anonymization) 
for teaching development and research purposes also.
